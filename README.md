I-Stream
I-Stream is a front-end movie listing and streaming web application built without any backend. It allows users to create an account, browse a catalogue of films, view movie details, and watch them directly in the browser using a custom HTML5 video player. User sessions, watchlists, and watch history are persisted locally in the browser via localStorage.
